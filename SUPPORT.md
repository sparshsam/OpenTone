# OpenTone Support

## Where to get help

- **Documentation** — See [docs/](docs/) for architecture, product spec, and positioning.
- **Bug reports** — Open a [GitHub Issue](https://github.com/sparshsam/OpenTone/issues/new).
- **Feature requests** — Open a [GitHub Issue](https://github.com/sparshsam/OpenTone/issues/new) with the `enhancement` label.
- **Discussions** — Use [GitHub Discussions](https://github.com/sparshsam/OpenTone/discussions) for questions and community help.

## Before opening an issue

1. Check [existing issues](https://github.com/sparshsam/OpenTone/issues) for duplicates.
2. Search [docs/](docs/) for relevant guidance.
3. Confirm you are on the latest version.

## Response times

This is a side project. Issues are typically addressed within a week.

# Age Rating Notes — OpenTone

Prepared for store age-rating questionnaires (Microsoft Store, Apple Mac App Store, etc.).

## App Overview

OpenTone is a desktop music player for locally owned audio files. It has no server, no online content, no social features, and no user-generated content that is visible to others.

## Store Questionnaire Answers

## Content Description

The app plays audio files that the user provides. OpenTone does not create, host, or distribute music. The user controls all content.

- **Music playback**: User-provided local audio files only
- **Visual content**: Album artwork (user-provided), text-based track lists, settings UI
- **No user-generated content shared**: No content leaves the device
- **No text chat or messaging**: No communication features
- **No web browser or web view**: The app embeds no browser engine

## Important Caveats

1. **Store ratings are at the store's discretion.** These notes are advisory. The final age rating is determined during store submission.
2. **Album artwork** displayed in the app comes from the user's music files. OpenTone does not moderate this content. Standard store disclaimers about user-provided content may apply.
3. **Future versions** may add features that affect age rating. Each update should re-evaluate these answers.
